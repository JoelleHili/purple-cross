# Shireburn Vue Interview Test

This repository contains a Vue.js project created as part of an interview test for **Shireburn Ltd**. The main task involves using the provided JSON file to display or manipulate its data within the application.

---

## Project Setup

### Prerequisites

* [Node.js](https://nodejs.org/) (v18+ recommended)
* [npm](https://www.npmjs.com/) (v9+ recommended)

### Install Dependencies

```bash
npm install
```

### Run the Application

```bash
npm run dev
```

Open your browser and navigate to `http://localhost:5173` (or the port shown in the terminal) to view the app.

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
